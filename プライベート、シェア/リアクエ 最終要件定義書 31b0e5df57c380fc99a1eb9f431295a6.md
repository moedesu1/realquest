# リアクエ 最終要件定義書

・第四境界のguest ARGという個人で出品してるコーナーのようなサイトを作りたい。

・このサイトの動線を参考にして。

・このサイトの動線商品(guestARGの商品が有料になったイメージ)

[https://shop.daiyonkyokai.net/](https://shop.daiyonkyokai.net/)

このサイトはミステリックなので、背景をアンティークな本屋(あるいはゲーム屋)みたいにして、多ジャンル配列できるようにしたいです。

製作者やジャンル、プレイ人数、プレイ時間 がパッと分かる。

そして、お問い合わせ先として、クリエイターのSNSやメールを載せておいてやり取りできる形。

ホントはゲーム屋がいいんですが、 とりあえずアンティークな本屋で色んなジャンルの本が売ってるイメージ。
